## ✨ Features / 功能特性

### 🇬🇧 English
- **"Crate" Style Organization**: Packages fragmented, messy browser bookmarks into clean, highly scannable grid blocks (crates).
- **Makefile Automation**: Built-in simple command pipeline to compile, build, or deploy your static link repository effortlessly.
- **Privacy & Lightning Fast**: Pure static frontend architecture with zero heavy tracking scripts, ensuring snappy-fast loading across all devices.

### 🇨🇳 中文
- **“集装箱”式收纳**：将零散无序的浏览器书签转化为分门别类、一目了然的响应式网格导航“方块”，告别杂乱。
- **Makefile 自动化驱动**：内置底层 Make 工作流，一键自动更新链接数据、渲染页面或打包本地静态网页。
- **纯净与闪电加载**：纯静态轻量化设计，无臃肿的前端框架与跟踪脚本，实现多端无缝秒级加载。

## 🚀 Automation Commands / 自动化指令

You can manage this navigation site easily via the terminal using built-in commands: / 您可以通过项目内置的 Make 指令流水线轻松管理和构建本导航站：

- `make build` : Compile and regenerate the static HTML dashboard pages. / 自动编译并生成最新的静态导航网页。
- `make serve` : Preview the bookmark navigation dashboard locally. / 在本地启动轻量服务，实时预览导航站效果。
