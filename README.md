## ✨ Features / 功能特性

- 🦀 **Rust High Performance**: Blazing-fast `minimal_server` backend providing lightning-fast rendering. / 纯 Rust 后端服务支撑，内存占用极低。
- 💾 **SQLx Persistence**: SQLite storage with built-in database migration schemas. / 采用 SQLx 异步持久化存储，内置规范的数据迁移管理。
- ⚙️ **Makefile Automation**: Complex pipeline compilation completely short-cutted into simple make actions. / 繁琐的 Cargo 与环境配置已被 Make 流水线一键封装。
